pyracmon.graph.identify¶

class pyracmon.graph.identify.IdentifyPolicy(identifier)[source]¶

Provides functionalities to handle identical entities in appending sessions of a graph.

get_identifier(value)[source]¶

Returns identification key from an entity value.

Parameters

value (object) – An entity value.

Returns

Identification key of the entity value.

Return type

object

identify(prop, candidates, ancestors)[source]¶

Select parent nodes and identical nodes of a new entity.

Parameters

  • prop (GraphTemplate.Property) – Template property for new entity.

  • candidates ([Node]) – Nodes having the same identification key.

  • ancestors ({str: [Node]}) – Mappings from property name to identical node set which appeared in current appending session.

Returns

  • [Node | None] – Parent nodes which the node of new entity should be appended newly. None means to append a new node without parent.

  • [Node] – Identical existing nodes propagated to identifications of children as ancestors.

class pyracmon.graph.identify.HierarchicalPolicy(identifier)[source]¶

Default policy for template properties having identifier.

This policy identifies nodes whose entity has the same identification key as one of appending entity and whose parent is also identical to the parent of the entity.
