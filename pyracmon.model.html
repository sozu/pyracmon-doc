pyracmon.model¶

pyracmon.model.define_model(table, mixins=[])[source]¶

Create a model type representing the table.

When the same attribute is defined in multiple mixin types, the former one has a priority.

Parameters

  • table (Table) – Schema of table.

  • mixins ([type]) – Types the created model type inherits.

Returns

Created model type.

Return type

type
