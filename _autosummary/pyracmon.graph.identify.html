class pyracmon.graph.identify.IdentifyPolicy(identifier)[source]¶

Provides entity identification functionalities used during appending entities to a graph.

Identification mechanism is based on the equality of identification keys extracted by entities.

identify(prop, candidates, ancestors)[source]¶

Select parent nodes and identical nodes of a new entity.

This method is called during appending an entity to a graph.

Parameters

  • prop (GraphTemplate.Property) – Template property for new entity.

  • candidates (List[Node]) – Nodes having the same identification key as the key of new entity.

  • ancestors (Dict[str, List[Node]]) – Parent nodes mapped by property names.

Returns

The first item is a list of Parent nodes which the node of new entity should be appended newly. None means to append a new node without parent. The second item is a list of identical nodes, which will be merged into ancestors and used in subsequent identifications of child entities.

Return type

Tuple[List[Optional[Node]], List[Node]]

class pyracmon.graph.identify.HierarchicalPolicy(identifier)[source]¶

Default identification policy used in a container where identification function is defined.

This policy identifies nodes whose entity has the same identification key as the key of appending entity and whose parent is also identical to the parent of the entity.
