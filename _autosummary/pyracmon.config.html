pyracmon.config¶

This module exports types and functions for configurations.

PyracmonConfiguration is a class exposing configurable attributes. An instance of the class held in this module is treated as global configuration. pyracmon is an only awy to change it. Changes done in with block is activated after the block finished without error.

>>> assert default_config().name == "default"
>>> with pyracmon() as cfg:
>>>     cfg.name = "my_config"
>>> assert default_config().name == "my_config"

See the attributes of PyracmonConfiguration to know all configuration keys and their effects.

A class to hold all configurable values in attributes.

Parameters

  • name (str) – Name of this configuration. This value has no effect on any behavior of modules. Default is default .

  • logger (Union[str, logging.Logger]) – Logger or the name of logger used for internal logs such as query logging. Defalut is None .

  • log_level (int) – Logging level of internal logs. Default is logging.DEBUG .

  • sql_log_length (int) – Maximum length of query log. Queries longer than this value are output being trimmed. Default is 4096 .

  • parameter_log (bool) – Flag to log query parameters also. Default is False .

  • paramstyle (str) – Parameter style defined in DB-API 2.0. This value overwrites the style obtained via DB module. Default is None .

  • type_mapping (Callable[[str], type]) – Function estimating python type from type name in database and optional DBMS dependent keys. Default is None .

  • graph_spec (pyracmon.model_graph.ConfigurableSpec) – Graph specification used as default. Default is None .

  • fixture_mapping (Callable[[pyracmon.model.Table, pyracmon.model.Column, int], Any]) – Function generating fixture value for a column and an index. Default is None .

  • fixture_tz_aware (bool) – Flag to make fixture datetime being aware of timezone. Default is True .

  • fixture_ignore_fk (bool) – Flag not to generate fixuture value on foreign key columns. Default is True .

  • fixture_ignore_nullable (bool) – Flag not to generate fixuture value on nullable columns. Default is True .

  • timedelta_unit (Dict[str, Any]) – Default keyword arguments to pass datetime.timedelta() used in near matcher.
