class pyracmon.marker.Marker[source]¶

This class provides the abstration mechanism for marker creation used to embed parameters in a query.

DB-API 2.0 defines some paramstyle s and each one corresponds to a class which inherits this class. Due to them, you just use a unified marker $_ in most cases despite which DB driver you are using.

Therefore, documentations of this class does not help application coding. They are written for library developers.

class pyracmon.marker.NumericMarker(start=0)[source]¶

This marker renders :x for the parameter at index x (starting from 1).

Automatic numbering is used when invoked with no key , otherwise specified position is selected. The index obtained by next automatic numbering is an incremented value of the last obtained index.

class pyracmon.marker.NamedMarker(start=0)[source]¶

This marker renders :key for the parameter bound to key .

Automatic key selection is used when invoked with no key , otherwise specified key is selected. Each automatic key is in the form of keyn where n is specified index or its order in generated keys.
