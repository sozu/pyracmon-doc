classmethod fixture(db, variable=None, index=None, cfg=None)[source]¶

Inserts record(s) with auto-generated column values.

:param : :type : param db: DB connection. If a value evaluated to be False in boolean context, generated model is not inserted and just returned. :param : When dict, model object or a list of them, inserts record(s) represented by them. Unspecified column values are generated. :type : param variable: When int, inserts records as many as the number. All of their column values are generated. :param : :type : param index: Use this to specify index used to generate column values explicitly. If set, indexing state is not updated. :param : This argument is prepared only for internal use and can be changed or removed in future version. :type : param cfg: Configuration used to control the generation of fixuture values. :param : :type : returns: Inserted model(s).
