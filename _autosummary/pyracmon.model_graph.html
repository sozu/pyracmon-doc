pyracmon.model_graph¶

This module provides graph specifications to deal with model types.

Most of them are not used directly except for ConfigurableSpec which is a configurable attribute of PyracmonConfiguration .

class pyracmon.model_graph.ConfigurableSpec(*args)[source]¶

Extension of GraphSpec prepared to integrate model types into graph specification.

This class exposes additional configurable attributes which controls the graph operation for model types. In global configuration, graph_spec attribute is an instance of this class, thus changes on it changes graph operations on model types.

Note

The implementation of this class is not stable, don’t depends on it.

Parameters

include_fk (bool) – Determines whether including foreign key columns in the result of graph serialization.
