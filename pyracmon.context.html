pyracmon.context¶

class pyracmon.context.ConnectionContext(identifier=None, **configurations)[source]¶

This class represents a context where DB operation is done on the configuration it has.

By default, each instance has the local copy of global configuration which can be changed via configure()

configure(**configurations)[source]¶

Change the configuration of this context.

Parameters

configurations ({str: object}) – Pairs of configuration name and its value.

Returns

This instance.

Return type
