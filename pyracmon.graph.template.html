pyracmon.graph.template¶

class pyracmon.graph.template.GraphTemplate(definitions)[source]¶

This class specifies the structure of a graph.

Use GraphSpec.new_template() to construct the instance of this class.

Each instance is composed of template properties each of which corresponds to a node container of a graph. Applying shift operator to properties creates the parent-child relationship between them and it corresponds to an edge of a graph.

For example, next code creates a template composed of 4 properties in which d is a child of c, and b and c are children of a.

>>> template = spec.new_template(a = int, b = str, c = int, d = float)
>>> template.a << [template.b, template.c]
>>> template.c << template.d
